差分约束

发表于2022-05-14|更新于2022-05-15|算法图论差分约束
|阅读量:

作用

求不等式组的可行解

思路

对于一组xi<=xj+ck，考虑最短路过程中的三角不等式，若u->v，一定存在dis[v]<=dis[u]+w(如果不成立，dis[v]应该被dis[u]更新)，所以我们可以将最短路和不等式组联系起来，将每个未知数对应图的一个顶点，把所有不等式化作图中的边。

做法

  1. 先将每个不等式xi<=xj+ck，转化成一条从xj走向xi，长度为c的一条边
  2. 找一个超级源点，从该点出发可以到达所有的点（保证所有的条件都被用到）
  3. 从原点求单源最短路（最长路也可以，但要构造xi>=xj+ck）
  4. 结果1：若存在负环，无解。
  5. 结果2：没有负环，dis[]就是一组可行解

Ps：最长路就是正环

负环无解的证明：

x[2]≤ x[1]+c[1]
     ...
x[k]≤ x[k-1]+c[k-1]
x[1]≤ x[k]+c[k]

x[2] ≤ x[1]+c[1]
     ≤ x[k]+c[k]+c[1]
     ≤ x[k-1]+c[k-1]+c[k]+c[1]
     ...
     ≤ x[2]+c[2]+...+c[k-1]+c[k]+c[1]
因为是负环，c[2]+......c[k]+c[1]<0
所以不成立

求每个变量的最值

结论：求最小值，跑最长路，求最大值，跑最小值

以求最大值为例，跑最短路

{x1,x2,…,xn} <=> {x1+d,x2+d,x3+d…xn+d}

因此想要有最大值一定有一个条件，绝对关系：xi<=c[i],xi>=c[i]，xi=c[i];

对于xi<=c 转化：建立超级源点，0， 0->xi(c)
